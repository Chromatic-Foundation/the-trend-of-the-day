# Privacy Policy – The Trend of the Day

**The Trend of the Day** does not collect personal data.  
The only data stored is the city name entered by the user for weather updates, which is saved locally in the browser.  

**API usage:**  
- WeatherAPI → retrieves city weather  
- YouTube API → retrieves trending video  
- Reddit API → retrieves top post  
- Binance API → retrieves crypto prices  

No personal, financial, or browsing history data is collected or shared.  
No data is sold or transferred to third parties.  

By using this extension, you consent to these terms.
